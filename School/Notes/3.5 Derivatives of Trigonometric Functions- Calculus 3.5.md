# Derivative of other Trig function

$$\frac {d}{dx}  tanx \text{ = }sec^{2}x$$
$$\frac {d}{dx} secx = (secx)(tanx)$$
$$\frac {d}{dx} cotx = -(csc^2x)$$
$$\frac {d}{dx} secx = (-cscx)(cotx)$$
