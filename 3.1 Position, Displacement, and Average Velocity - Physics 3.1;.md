

# Average Velocity
$$\overline v = \frac{\Delta x}{\Delta t} = \frac {x_{2} - x_{1}}{t_{2}-t_{1}}$$
# Displacement

If an object goes thru a series of displacement then _total displacement_:



$$\Delta x_{total}= \sum\limits_{i}x_i$$

# Distance
Total distance: 
$$\sum_{i}\vert\Delta x_{i}\vert$$
